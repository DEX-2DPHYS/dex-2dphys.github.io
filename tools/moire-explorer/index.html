Moire Explorer

Pedagogical white-background explorer for moiré superlattices in graphene, hBN, MoS₂, MoTe₂, WS₂, WTe₂, SrTiO₃, and BaTiO₃. Supports direct atom view, coincidence-site lattice (CSL), up to four stacked layers, per-element color schemes, square or hexagonal unit cells, switchable perovskite AO/TiO₂ surface termination and planes, and soft random strain displacement maps. SI units are used throughout (m, rad).

Gaps are typed as adjacent surface-to-surface spacings. Layer origins are then reconstructed from the visible atomic thickness of each material, which prevents thicker TMD layers from inheriting the same incorrect center-to-center spacing as graphene.
Each z-relax button moves the layer above that interface against the full stack energy over the current vdW scan range.
CSL keeps its own tile grid. The vdW interface scan and the A → B path use separate worker settings, so the interface scan can run on 1 worker while the path requests any explicit number of Web Workers. Pair-interface and fixed-gap path evaluation use the path workers; full-stack per-layer relaxation remains iterative per step.

vdW / LJ

Uses the selected CSL pair and the typed adjacent surface gaps. Lorentz-Berthelot mixing is applied for σ, geometric mixing for ε, perovskites are extended to the cutoff depth, and an attractive -C₆/r⁶ continuum tail can be added beyond the explicit slab.

Path Scan

Records the current stack pose for all layers using only rotation and x/y translation. Intermediate poses use shortest-angle interpolation for θ and linear interpolation for translation, so steps 1 and N match A and B exactly. Ordered mode evaluates endpoint-inclusive steps. Random mode samples random interpolation parameters p in [0,1], sorts each completed point into the curve, and keeps running until you stop it. Full-stack z-relax remains ordered-only.

Layer Stack (max 5)

Model notes: CSL points are marked when the inter-layer in-plane distance satisfies |r₁-r₂| < r_tol. Small-angle moiré estimate uses L_m ≈ ā / √(δ² + θ²), with θ in radians and δ lattice mismatch. STO is modeled as cubic SrTiO₃ with a = c = 3.905 Å, and BTO as room-temperature tetragonal BaTiO₃ with a = 3.994 Å and c = 4.036 Å, rendered as alternating AO and TiO₂ planes with selectable top-surface termination. The vdW module performs a sampled explicit 12-6 Lennard-Jones sum over the selected pair, using typed adjacent surface gaps rather than one global center spacing, extends STO/BTO bulk to the cutoff along the surface normal, and can add a continuum attractive tail beyond that depth. The A → B path scan records two in-plane stack poses and evaluates the same energy model along shortest-angle rotation plus linear translation interpolation. Strain displacement in each crystal uses soft random sinusoidal fields u(r) with tunable wavelength and intensity (m).
