What is this? The target the electrons scatter in — a resist (PMMA) or substrate/metal (Si, GaAs, Au). Each carries an atomic number \(Z\), atomic weight \(A\), and mass density \(\rho\).
Physics behind Elastic scattering strength grows with \(Z\) (heavy atoms deflect electrons more and backscatter strongly), while penetration falls with density. The continuous-slowing-down (CSDA) range scales as \[ R \approx \frac{K}{p_\text{CSDA}}\,\frac{A\,E^{1.67}}{Z^{0.89}\,\rho}. \]
How to use it Pick PMMA to mimic resist exposure, or Au to see a strong backscatter halo (the proximity effect). Heavy, dense targets give shallower, wider interaction volumes.
What is this? The primary beam energy \(E_0\) of the incident electrons.
Physics behind Penetration rises steeply with energy — the Kanaya–Okayama-style range goes as \(R \propto E^{1.67}\). Higher energy also scatters more in the forward direction, since the characteristic screening angle \[ \theta_0 = \frac{k\,Z^{1/3}}{E\,\beta}, \qquad \beta=\sqrt{1-\gamma^{-2}},\ \ \gamma=1+\frac{E}{511\,\text{keV}} \] shrinks as \(E\) grows.
How to use it 1–200 keV (typical EBL is 10–100 keV). Raise it to bury the interaction volume deeper and spread the long-range proximity dose.
What is this? The number of Monte-Carlo electron histories actually simulated.
Physics behind Every trajectory is one random walk; the deposited-energy map is their sum. Statistical noise in each dose cell falls as \[ \sigma \propto \frac{1}{\sqrt{N}}. \] Averaged quantities (backscatter fraction, range) converge as \(N\) increases.
How to use it 1000 is fast; use 5000–20000 for a smooth heatmap. All \(N\) contribute to the dose, not just the drawn ones.
What is this? How many trajectories are drawn in the left (track) view.
Physics behind Purely a visualisation choice — it does not affect the physics or the dose statistics, which always use all \(N\) histories.
How to use it Keep it small (50–200) so individual paths stay readable; set it to 0 to skip drawing entirely and run faster.
What is this? The integration step length \(\Delta s\) taken along each trajectory.
Physics behind Between elastic collisions the electron moves in straight free flights sampled from an exponential distribution, and energy is lost continuously along the way. Each move advances by \(\min(\Delta s,\ \text{flight})\), and the energy update integrates \[ E(s) = \left(E_0^{1.67} - C\,s\right)^{1/1.67}. \] Smaller \(\Delta s\) resolves the energy loss and curvature more accurately.
How to use it 1–5 nm is a good balance. Large steps run faster but coarsen the tracks and the dose.
What is this? The depth (vertical size) of the simulated and plotted domain. It also sets the lateral field of view, locked equal to it.
Physics behind To capture the whole interaction volume the depth must exceed the CSDA range \(R\); otherwise deep tracks are clipped. The panel warns when depth \(< 1.1\,R\) and suggests \(\approx 1.3\,R\).
How to use it Follow the suggestion under the controls — set it a little above the CSDA range for the current energy and material.
What is this? The lateral width of the domain. It is locked equal to the depth.
Physics behind Keeping the horizontal and vertical scales equal (a square domain) shows the interaction volume with no aspect-ratio distortion, so lateral spreading is read off directly.
How to use it Read-only — change Depth and this follows.
What is this? The grid resolution (cells per side) of the deposited-energy histogram.
Physics behind Dose is binned onto a \(W\times H\) grid with \(W=H=\) this value. Finer grids resolve sharper features but put fewer electrons in each cell, so noise per cell rises (recall \(\sigma\propto 1/\sqrt{N}\)).
How to use it 128–256 is typical; raise it together with the number of trajectories to keep the map clean.
What is this? A tuning knob that scales how frequently elastic (direction-changing) collisions happen.
Physics behind It rescales the elastic mean free path \[ \lambda = \frac{\lambda_0\,E^{p}}{Z^{q}\,\rho^{r}}\cdot\frac{1}{m}, \qquad (p,q,r)=(1,\,0.8,\,0.6). \] A larger multiplier \(m\) shortens \(\lambda\), giving more frequent large-angle deflections and more diffusive, backscattered paths.
How to use it This is a toy-model calibration knob. Increase it to exaggerate scattering and backscatter; decrease it toward ballistic behaviour.
What is this? A prefactor that scales the continuous energy-loss (stopping power).
Physics behind It multiplies \(dE/ds\), and the range is inversely proportional to it: \[ R \propto \frac{1}{p_\text{CSDA}}. \] The model is tuned so \(p_\text{CSDA}\approx 0.9\) reproduces \(\sim 80\ \mu\text{m}\) at 100 keV in Si.
How to use it Nudge it to match a known range: larger values shorten the range (shallower volume), smaller values deepen it.
What is this? Switches off elastic scattering so electrons travel in straight lines (a pure CSDA test).
Physics behind With no angular deflection, each track is a straight line whose length equals the CSDA range \(R\). This isolates the slowing-down physics from scattering.
How to use it Enable it to check that the straight-line penetration matches the predicted CSDA range, then turn it off for realistic spreading.
What is this? Run starts the Monte-Carlo simulation, Stop halts it mid-run, and Export PNG saves the two views side by side.
Physics behind Run launches \(N\) electron histories, accumulating both the drawn trajectories and the deposited-energy grid; the progress bar tracks completion.
How to use it Adjust the inputs, press Run, and use Stop if a large run is taking too long. Export saves a snapshot for figures.
What is this? The backscattered fraction — electrons that leave back through the entry surface.
Physics behind Counted whenever a trajectory returns to \(z<0\). The backscatter coefficient \(\eta\) rises with atomic number \(Z\); these electrons carry energy far from the beam and drive the long-range proximity effect.
How to use it Compare across materials — Au backscatters far more than PMMA.
What is this? The mean straight-line distance from entry to each electron's final stopping point, \(\langle\sqrt{x^2+y^2+z^2}\rangle\).
Physics behind Because of scattering detours this end-to-end distance is generally shorter than the CSDA path length \(R\); the gap grows with more scattering.
How to use it Watch how it drops below the CSDA range as you increase the elastic multiplier.
What is this? The average of each electron's maximum penetration depth in \(z\).
Physics behind Reflects how deep the interaction volume reaches; it approaches the CSDA range for forward-peaked (high-energy, straight) transport and falls when scattering randomises directions.
How to use it Use it to check that the chosen Depth is large enough to contain the tracks.
What is this? The trajectory view — individual electron paths projected onto the \(x\)–\(z\) plane.
Physics behind Each coloured line is one random walk: straight free flights punctuated by elastic deflections, ending when the electron stops (\(E<0.05\) keV) or backscatters (\(z<0\)). Equal \(x\)/\(z\) scaling shows the true shape of the interaction volume.
How to use it Read off penetration depth, lateral spread, and backscattering at a glance; limit the number of drawn tracks to keep it legible.
What is this? The deposited-energy (dose) view — a 2D histogram of energy left in the material.
Physics behind Each step deposits its energy loss \(dE\) into the grid cell it passes through. This dose \(D(x,z)\) is the physically relevant quantity: it exposes the resist and, via its long tail, produces the proximity effect.
How to use it Use the controls below to bring out either the bright core or the faint long-range tail.
What is this? The colour map applied to the normalised dose.
Physics behind Purely a display transform, mapping dose \(\to\) colour. Perceptually-uniform maps (Viridis, Inferno, Turbo) represent equal dose differences as equal perceived colour differences.
How to use it Viridis is a safe default; Grayscale is handy for figures or printing.
What is this? A lower dose cutoff, set on a logarithmic slider from \(10^{-8}\) up to \(0.95\) of the maximum.
Physics behind Deposited energy spans many orders of magnitude. Everything below \(D_\text{min}=f\cdot D_\text{max}\) is treated as the floor, which lets you hide noise or reveal the faint backscatter tail.
How to use it Lower it to expose the long-range proximity dose; raise it to isolate the bright core near the beam.
What is this? Caps the top of the colour scale at a chosen percentile of the dose values.
Physics behind A few cells right under the beam can be far brighter than the rest. Clipping the scale at, say, the 99.5th percentile stops those hot spots from saturating the map and washing out structure.
How to use it Lower the percentile to boost mid-level contrast; raise it toward 100% to keep the true peak.
What is this? A gamma (tone) curve applied to the normalised dose.
Physics behind Applies a nonlinear mapping \[ t \mapsto t^{\,1/\gamma},\qquad t\in[0,1], \] which brightens faint features for \(\gamma>1\) and compresses them for \(\gamma<1\).
How to use it Increase \(\gamma\) to pull up the dim tail; decrease it to emphasise only the strongest dose.
What is this? Maps colour to the logarithm of the dose instead of a linear scale.
Physics behind Because dose varies over several decades, a log mapping \(\ (\log D)\ \) shows both the intense core and the weak long-range tail in one image.
How to use it Turn it on to see the full dynamic range at once; combine with the cutoff to set the floor.
What is this? Makes everything below the cutoff fully transparent instead of colouring it as the map's zero.
Physics behind Acts like a binary exposure threshold — a crude analogue of a resist that only clears above a critical dose \(D_\text{min}\).
How to use it Enable it with the cutoff to visualise which region would be "developed" at a given clearing dose.
What is this? Overlays iso-dose contour lines on the heatmap.
Physics behind Lines of constant dose \(D(x,z)=\text{const}\) (drawn by marching squares) trace the shape of the exposure — analogous to the developed-resist boundary at a given clearing dose.
How to use it Turn it on to compare the extent of different dose levels; off for a cleaner image.
